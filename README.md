## Overview
Bijlee is a mobile application designed to provide on-demand, location-based EV charging
services for electric vehicle owners in Toronto.

The app enables users to request, schedule, track, and pay for EV charging while allowing
technicians and administrators to manage service delivery efficiently.


## Problem Statement
EV owners face limited access to public charging stations, long wait times, and inconvenient
locations. Bijlee solves this by delivering charging services directly to the user's location.

## Key Features

- Real-time service tracking
- Notifications and order updates
- Secure payments and subscription billing
- Technician job management workflows
